## 発表の概要

「BlackBox モデルの説明性・解釈性技術の実装」

表形式データ、画像データ、テキストデータに対して、種々の説明性・解釈性技術を実装し、実行してみた結果や感想について解説します。を紹介します。具体的にはBERTと呼ばれる自然言語処理のディープラーニング・モデルを利用し、教師あり学習の推論結果を説明する手法を紹介します。
